homport by schworer

Homport makes it easier to interact with Houdini nodes in Python. Run it during interactive sessions to quickly wire up nodes, see the values of parameters, and more.

my_node = hou.node('/obj').createNode('geo')
another_node = hou.node('/obj).createNode('null')
my_node >> another_node

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/schworer/homport
